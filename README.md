# bookshelfAPI

<h3>API dapat menyimpan buku</h3>
#)API harus dapat menyimpan buku melalui route:<br/>
  Method : POST<br/>
  URL : /books<br/>
  Body Request:<br/>
```javascript
    {
        "name": string,
        "year": number,
        "author": string,
        "summary": string,
        "publisher": string,
        "pageCount": number,
        "readPage": number,
        "reading": boolean
    }
```
Objek buku yang disimpan pada server harus memiliki struktur seperti contoh di bawah ini: <br/>
```
git status
git add
git commit
```
